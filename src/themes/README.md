## Folder `themes`

Folder `themes` digunakan untuk menyimpan themes atau custom style pada component-component react.

Aturan penggunaan folder themes

```
├── theme
│   ├── default/
│   │   ├── index.scss
│   │   ├── base.scss
│   │   ├── colors.scss
│   │   ├── utils.scss
│   │   └── override/
│   │       └── button.scss
│   └── dark/
│       ├── index.scss
│       ├── base.scss
│       ├── colors.scss
│       ├── utils.scss
│       └── override/
│           └── button.scss
```
